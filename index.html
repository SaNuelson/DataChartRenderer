CSV2GC - Data Chart Renderer

Data Chart Renderer (DCR) is a jQuery plugin that allows you to easily render charts using raw data as an input along with options.
The plugin is divided into multiple layers. Depending on your needs, you can include specific subset of the plugin to suit your needs.
