Welcome to CSV2GCR app that enables you to render data as charts via Google Charts app.
In the top left corner there's a menu that handles most of the functionality nicely packed away.
File enables you to work with your data files, Chart helps you exporting the configured chart data,
and finally Help enables you to quickly hide or display different parts of this website including this help section.
Load the data file you want to visualize.
As you can see from the name of this app, we currently only support CSV files.
Don't you worry, we'll fix that sometimes... maybe.
Anyways, load it up in the top left menu: File > Load New Datafile
Now that you've loaded up your data, a table should show up below taking the first 5 rows.
Just to give you an idea what you're working with.
Next, select a specific type of chart you'd like to visualize.
You can find it in the top right corner. Here, let me show you.
You're doing great! If you did all according to plan, below the table a configuration section should show up.
This is where you put in all the info about how you want your chart displayed.
A single option consists of the following parts:
  • Label describing which "aspect" of the chart it's bound to.
  • Select element where you choose which column of your data corresponds with it.
  • Select element where you choose what type is your data / what type you want it to be used as.
  • Text input element for when you want to specify a format. This is generally necessary only for weird date/datetime formats.
Furthermore, some options are optional, those you can leave blank if you'd prefer. Others can be specified multiple times. Those can be copied.
